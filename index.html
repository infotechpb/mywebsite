Class 9 - Complete Syllabus

Website Design (Full Course)

Topics covered:

  • HTML5 semantic elements & forms
  • CSS3 – Flexbox, Grid, Animations
  • Responsive Web Design (Media Queries)
  • JavaScript basics – DOM, events, validation
  • Introduction to Bootstrap / Tailwind
  • Hosting a website (GitHub Pages, Netlify)

Database Management System (DBMS)

  • File system vs DBMS, advantages
  • ER diagrams, relational model
  • SQL – DDL, DML, DCL, Joins, Subqueries
